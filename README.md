## プロジェクトについて

フィボナッチ数列を返す API

### 環境

| 言語・フレームワーク  | バージョン |
| --------------------- | ---------- |
| Python                | 3.11.4     |
| Django                | 5.0.1      |
| Django Rest Framework | 3.15.1     |

### ローカルでのプロジェクトの作成と起動

```
pip install requirements.txt
```

```
python3 manage.py runserver
```

### 動作確認

以下の URL にアクセスする<br>
https://fibonacci-api-jozr.onrender.com/fib?n=100<br>
